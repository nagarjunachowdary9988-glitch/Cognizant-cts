# Week 2 Quiz – C# Programming

## 1. What are Generics?

Generics allow you to write reusable, type-safe code by using type parameters instead of specific data types.

---

## 2. What are the advantages of Generics?

* Type safety
* Code reusability
* Better performance
* Reduced code duplication

---

## 3. What is Exception Handling?

Exception handling is a mechanism used to handle runtime errors and prevent abnormal program termination.

---

## 4. Which keywords are used in Exception Handling?

* try
* catch
* finally
* throw

---

## 5. What is LINQ?

LINQ (Language Integrated Query) is a feature in C# used to query and manipulate data from collections, databases, XML, and other data sources.

---

## 6. What are the advantages of LINQ?

* Simple syntax
* Readable code
* Type safety
* Better productivity

---

## 7. What is a Delegate?

A delegate is a type-safe reference that holds the address of a method.

---

## 8. What is an Event?

An event is a mechanism that allows one object to notify other objects when a specific action occurs.

---

## 9. What is Unit Testing?

Unit testing is the process of testing individual units or methods of a program to ensure they work correctly.

---

## 10. What is Logging?

Logging is the process of recording application events, errors, and execution details for debugging and monitoring.

---

## 11. What is NuGet?

NuGet is the package manager for .NET used to install, update, and manage libraries and dependencies.

---

## 12. What are the benefits of NuGet?

* Easy package management
* Automatic dependency resolution
* Version control
* Faster development
